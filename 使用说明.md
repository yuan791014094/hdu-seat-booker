# 杭电自习室抢座

## 打开网页

第一次使用先复制模板：

```powershell
copy config.example.yaml config.yaml
copy booking.example.json booking.json
```

然后在 `config.yaml` 填杭电账号密码，在 `booking.json` 填候选座位。

双击 `打开网页.bat`，浏览器会打开：

`http://127.0.0.1:8765`

如果打不开，在本目录手动运行：

```powershell
python web_app.py
```

## 日常怎么用

1. 在网页最上面的 `booking.json` 里改候选座位。
2. 点 `保存 booking.json`。
3. 点 `验证座位配置`，确认能解析到实际 `seat_id`。
4. 要马上预约就点 `立即按 JSON 预约`。
5. 要每天晚上 20:00 自动抢后天座位，就点 `安装每日自动抢座`。

## booking.json 填什么

主要改这几项：

```json
{
  "booking": {
    "date": "auto",
    "begin_hour": 12,
    "duration_hours": 10
  },
  "seats_priority": [
    {
      "room_name": "宋韵云图（四楼）",
      "seat_num": "96"
    }
  ]
}
```

- `room_name`：填网页里给出的房间名，默认推荐 `宋韵云图（四楼）`
- `seat_num`：填平面图上显示的座位号，例如 `96`、`395`
- `begin_hour`：开始小时，只能填整数小时，例如 `8`、`12`
- `duration_hours`：预约时长，例如 `10`
- `date`：建议保持 `"auto"`

`date: "auto"` 的规则：

- 晚上 20:00 前运行：约明天
- 晚上 20:00 后运行：约后天

## 查座位号

网页下方有 `座位和平面图`：

- 选择房间后点 `读取座位和平面图`
- 点击图上的座位点，会显示“显示座位号”和实际 `seat_id`
- 勾选 `显示座位号` 可以临时显示全部座位号

现在主要围绕四楼：

`宋韵云图（四楼）`

## 登录

如果提示登录失败，运行：

```powershell
python book.py --show-browser --dry-run
```

手动登录一次后会保存 `cookies.json`，之后通常可以复用。

## GitHub Actions

仓库不要上传真实的 `config.yaml`、`booking.json`、`cookies.json`。

如果要用 GitHub Actions 自动运行，在仓库 Settings -> Secrets and variables -> Actions 里添加：

- `HDU_USERNAME`：杭电账号
- `HDU_PASSWORD`：杭电密码
- `BOOKING_JSON`：完整的 `booking.json` 内容
